📚 Documentation

Welcome to the An RFID-Based Library Visitor Logging System! 📚 This Python desktop application, built with Tkinter, is designed to efficiently manage student records and track their library usage using RFID technology. It provides a simple yet powerful graphical user interface (GUI) for all your library management needs.

💻 Technologies & Libraries

  • Python 🐍: The primary programming language.
  • Tkinter: GUI toolkit used for the main interface, including ttk for themed widgets and messagebox for dialogs.
  • SQLite3: Lightweight, file-based database for storing students and logs.
  • reportlab: Generates PDF reports of library logs.
  • pandas: Reads and processes student data from Excel files for bulk import.
  • keyboard: Monitors RFID card scans and keyboard-related events.
  • Pillow (PIL): Displays and processes images in the GUI.
  • tkcalendar: Adds a user-friendly date picker widget.

📂 Core Functionalities

🛠️ System Management

  • User Authentication: Secure login 🔒 with predefined admin and staff credentials.
  • RFID Monitoring: A background thread listens for RFID scans to log student entry/exit.
  • GUI Management: Handles main window, menu bar, and navigation.

📝 Student Data Management

  • Add Student: Register a student with RFID, name, grade, and section.
  • View Students: Browse/search all registered students.
  • Update Student: Modify student details.
  • Delete Student: Remove student records.
  • Bulk Import: Import student records from Excel files.

🗃️ Database Management

  • Table Creation: Automatically sets up tables for students, logs, and purposes on first run.
  • Data Persistence: Ensures all records and logs are permanently saved.
  • Queries: Fetch specific student info, logs, or purposes efficiently.

📄 PDF Reporting

  • Log Generation: Create detailed PDF reports of student activity.
  • Custom Filters: Generate reports by student name or date range.
  • Automatic Saving: Reports are automatically saved to the desktop.

This system integrates RFID technology with modern Python libraries to deliver an efficient, secure, and user-friendly solution for library management.
